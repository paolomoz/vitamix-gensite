## Detailed Description

AEM Generative Websites is a developer demo tool that showcases real-time intent inference and AI-powered content generation for Adobe Experience Manager Edge Delivery Services.

**What it does:**

• Captures browsing signals on vitamix.com (page views, clicks, scroll depth, video engagement)
• Infers user intent profiles in real-time using weighted signal analysis
• Generates personalized search queries based on observed behavior
• Demonstrates AI-powered content generation via streaming blocks

**Key Features:**

Intent Inference Panel
- Live signal feed showing captured user interactions
- Real-time profile inference with confidence scoring
- Segment detection (new parent, gift buyer, upgrader, researcher)
- Purchase readiness and price sensitivity analysis

Content Generation
- Context-aware page generation using Claude + Cerebras
- Streaming block delivery via Server-Sent Events
- Reasoning visibility for AI decision transparency
- Self-improvement analysis with multi-agent consensus

Example Scenarios
- Pre-configured personas for testing and demos
- Short and long interaction sequences
- Realistic signal patterns for validation

**Intended Audience:**

This extension is designed for:
- Developers building on AEM Edge Delivery Services
- Teams exploring AI-powered personalization
- Internal demos and proof-of-concept testing

**Sites Supported:**

- vitamix.com (signal capture)
- *.aem.live (content generation POC)
